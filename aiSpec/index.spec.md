# Page Spec — Home (index.html)

## Page goal
Introduce me quickly and show my skills.

## Sections
1) Header (logo, name, nav links)
2) Hero:
   - Title: "Hi, I’m {Name}"
   - 1–2 sentences bio
   - 4 skill tags
3) Two cards:
   - Technical Skills (bullet list)
   - Soft Skills (bullet list)
4) Footer:
   - LinkedIn, GitHub, Email

## Styling
- Dark theme
- Cards with rounded corners
- Responsive grid
