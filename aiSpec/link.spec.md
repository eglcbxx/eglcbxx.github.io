# Page Spec — Links (links.html)

## Page goal
Share a small selection of external pages, tools, or resources that I personally like or recommend.

## Target audience
Anyone visiting the portfolio who wants to learn more about my interests or inspirations.

## Sections

### 1) Header
- Reuse the global header (logo, name, navigation links)

### 2) Page title
- Title: "Links"
- Short sentence explaining that these are recommended or favorite resources

### 3) Links section
Display two link cards.

Each link card must contain:
- Title of the resource
- One image (logo, screenshot, or placeholder)
- Short description explaining why this link is interesting or useful
- External link:
  - Opens in a new tab

### 4) Footer
- Reuse the global footer
- Social links: LinkedIn, GitHub, Email

## Content constraints
- Friendly but professional tone
- No promotional language
- Descriptions limited to 1–2 sentences

## Styling requirements
- Same card style as Portfolio page
- Responsive layout
- Visual consistency with other pages
